# 📄 PDF - Электрические схемы и документация ИЖ 2126

## Описание

Этот каталог содержит электрические схемы и техническую документацию ИЖ 2126 "Ода" в формате PDF. Здесь хранятся высокого качества отсканированные документы электрических систем и компонентов автомобиля.

## 📋 Содержание

- **2126-020.pdf** - Электрическая схема основной модели ИЖ-2126-020
- **21261-Fabula.pdf** - Полная техническая документация модели Fabula
- **21261-Fabula.jpg** - Электрическая схема модели Fabula (графический формат)
- **21261-Fabula_pr.pdf** - Электрическая схема модели Fabula (предварительный вариант)
- **S_priborkoy_2141.pdf** - Схема приборной панели 2141
- **Zad_dvornic.pdf** - Документация по задней части автомобиля
- **mont_blok.pdf** - Схема монтажного блока и электрооборудования

## 💡 Как использовать

1. Выберите необходимый документ из доступных файлов
2. Откройте PDF в Adobe Reader, браузере или другом просмотрщике PDF
3. Используйте функцию увеличения масштаба для лучшей читаемости схем
4. При необходимости распечатайте или сохраните копию для справки
5. Документы полезны при диагностике, ремонте и обслуживании электрооборудования

## 📚 Рекомендуемые источники

- **Основная документация**: ознакомьтесь с полной техдокументацией в формате PDF перед началом работ
- **Диагностика**: используйте схемы для проверки электрических цепей и соединений
- **Ремонт**: схемы помогут в правильной установке и подключении компонентов
