# 🖼️ JPG - Электрические схемы ИЖ 2126

## Описание

Этот каталог содержит электрические схемы ИЖ 2126 "Ода" в формате JPG (JPEG). Здесь хранятся графические изображения различных электрических систем и компонентов автомобиля.

## 📋 Содержание

- **2126-020.jpg** - Основная электрическая схема
- **21261-Fabula_pr.jpg** - Электрическая схема модели Fabula (предварительный вариант)
- **IZH-2126.jpg** - Схема ИЖ-2126
- **S_priborkoy_2141.jpg** - Схема приборной панели 2141

## 💡 Как использовать

1. Выберите необходимую схему из доступных файлов
2. Для лучшей читаемости рекомендуется открывать файлы в программе просмотра изображений с функцией масштабирования
3. При необходимости можно распечатать или использовать как справочный материал при ремонте и обслуживании
