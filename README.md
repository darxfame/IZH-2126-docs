# 🚗 Архив ИЖ 2126 «Ода» / IZH 2126 Archive

**Публичный репозиторий для документов по 2126. Коммиты приветствуются!**

## 🎯 О проекте
Этот репозиторий создан для централизованного хранения и сохранения информации об автомобиле ИЖ 2126 (известном также как «Ода») в интернете. С течением времени бумажные носители и старые онлайн-ресурсы исчезают, и мы хотим предотвратить утрату ценных технических данных, истории и культуры владения этим автомобилем.

## 📂 Что здесь можно найти
В рамках проекта собираются:
- 📘 Руководства по эксплуатации и ремонту
- 🔌 Электрические схемы и документация
- 🛠 Каталоги запчастей
- 📸 Фотоархивы и история модели
- 💬 Полезные материалы от сообщества (FAQ, тюнинг, восстановление)

## 🤝 Как внести вклад
Мы верим, что сохранение истории — это общее дело. Если у вас есть документы, сканы, фото или полезная информация:
1. **Сделайте Fork** этого репозитория.
2. **Добавьте файлы** в соответствующие папки (или создайте новые).
3. **Отправьте Pull Request** с описанием изменений.
4. Либо откройте **Issue**, если у вас есть вопросы или предложения.

Любая помощь в актуализации данных приветствуется!

## 📜 Лицензия
Материалы распространяются в целях сохранения культурного и технического наследия. Пожалуйста, соблюдайте авторские права при использовании документов.
