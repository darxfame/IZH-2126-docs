Это уже гораздо понятнее — речь не про конкретный бортжурнал DRIVE2, а про архивный/документационный проект по ИЖ 2126.

Тогда задача проекта выглядит так:

* собрать техническую документацию;
* сохранить исчезающие материалы;
* структурировать знания по платформе ИЖ 2126 / Ода / Фабула / 2717;
* сделать открытый community archive.

Идея очень сильная, потому что большая часть информации по ИЖ сейчас реально расползается по:

* старым форумам;
* DRIVE2;
* PDF/DJVU архивам;
* web archive;
* мертвым FTP;
* форумам москвичеводов;
* локальным сайтам начала 2000-х.
